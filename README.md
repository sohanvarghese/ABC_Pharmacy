# ABC Pharmacy Management System

## Technologies Used
- ASP.NET Core 8 Web API
- Vue 3
- Bootstrap 5
- Axios
- JSON File Storage

## Features
- Dashboard
- Medicine Management (CRUD)
- Search Medicines
- Low Stock Highlight
- Expiry Highlight
- Sales Management
- Sales History

## How to Run

### Backend

```bash
cd ABC_Pharmacy
dotnet run
```

### Frontend

```bash
cd pharmacy-ui
npm install
npm run dev
```
